# AI Workshop - OpenEdge ABL Project

An OpenEdge ABL project demonstrating modern architectural patterns including Business Entity, Singleton Factory, and layered architecture.

## Project Structure

- `src/business/` - Business Entity classes and dataset definitions
- `src/*.w` - UI Windows (Progress AppBuilder)
- `.windsurf/rules/` - Code style rules
- `.windsurf/workflows/` - Development workflows
- `doc/` - Architecture documentation
- `dump/` - Database schema definitions

## Prerequisites

- OpenEdge 12.8+
- Sports2000 database

## Getting Started

1. Create the sports2000 database: `ant db`
2. Open the project in PDSOE or your preferred ABL IDE
